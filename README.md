## Deployment

To deploy this project run

# Clone the repository
```bash
git clone  https://github.com/kumaratul31/capstone-project-ui.git
```

# Go into the repository
```bash
cd capstone2-project-ui
```


# Install dependencies
```bash
npm install
```


## Run Locally



```bash
  npm run dev
```


## From Browser UI
```bash
http://localhost:3000 

```

or
```
http://localhost:5173

```
